Leave Application

有給休暇申請

年次有給休暇・特別休暇（慶弔等）の申請手順です。

⏱所要時間：5分 📱システム：HR Core 📅申請期限：希望日の3営業日前まで
1🖥

System Login

HR Coreにログインし「勤怠申請」を選択

社内ネットワークまたはVPN接続で hr-core.company.jp にアクセス。左メニューから「勤怠申請」→「有給休暇申請」を選択します。

💡初めてのアクセスはITサポート（内線#3000）へ。シングルサインオン対応済みなので社員番号・パスワードでログインできます。
2📝

Input

取得希望日・休暇種別を入力する

「休暇日」「休暇種別（年次有給 / 半日有給 / 特別休暇等）」「業務引継ぎ先」を入力します。特別休暇は理由の選択が必要です。

📌 半日有給は「午前休」「午後休」から選択。午前休は〜12:00まで、午後休は13:00〜の出勤が必要です。
3✉️

Submit

申請を送信・上長承認を待つ

「申請する」ボタンで送信。上長にメール通知が届きます。承認・否認の結果はメールで通知されます。承認前に休暇取得してはいけません。

❌ NG：承認メールが届く前に休暇を取得すると「無断欠勤」扱いになる場合があります。必ず承認後に休暇を取得してください。
4✅

Confirmation

承認メール受信・休暇取得

承認メールが届いたら休暇の取得が確定です。チームメンバーへの事前連絡・引継ぎも忘れずに。Slackの「本日の勤務状況」チャンネルへの投稿も推奨します。

Overtime Application

残業申請

残業・時間外労働は原則事前申請が必要です。事後申請は月2回まで。

⏱所要時間：3分 📱システム：HR Core 📅申請期限：当日17時まで（事前）

平日残業

17:30以降の勤務。当日17時までに申請・承認が必要

休日出勤

土日・祝日の出勤。前週金曜17時までに申請

深夜残業

22時以降は深夜手当が加算（別途申告不要・自動計算）

月上限

45時間/月・360時間/年（労基法36協定）

1💬

Confirm with Manager

まず上長に口頭・チャットで相談・確認を取る

系統として「残業が必要な業務の発生」→「上長への相談」→「システム申請」の順で行います。上長の事前了承なく申請しても却下される場合があります。

2📝

Apply on HR Core

HR Coreで残業申請フォームに入力

「残業理由」「予定終了時刻」「対応業務内容」を入力。予定時刻を超える場合は延長申請が必要です（2時間単位）。

❌ NG：申請・承認なしで残業した場合、残業代が不支給になる場合があります。
3✅

Work & Record

承認後に残業開始・実績を打刻

承認メール受信後に作業を開始。終業時はHR Coreで打刻（PC操作で自動記録）してください。実際の終了時刻が申請時刻と2時間以上乖離する場合は翌営業日中に修正申請が必要です。

Marriage Procedure

結婚・氏名変更の手続き

婚姻届提出後2週間以内に、以下の手続きを行ってください。

1📋

Documents

戸籍謄本（抄本）を取得する

婚姻届受理後に市区町村役所で戸籍謄本（または抄本）を取得。新しい氏名・生年月日が確認できるものが必要です。

2📝

Apply

HR Coreで「身上変更届」を提出

氏名・住所（変更の場合）・通勤経路（変更の場合）を入力。戸籍謄本のスキャンをPDFで添付します。

📌 慶弔休暇（結婚：本人5日）もこのタイミングで申請できます。婚姻日から3ヶ月以内が申請期限です。
📌 結婚祝い金（10万円）も別途申請が必要です。「慶弔金申請」フォームから申請してください。
3🎫

ID Update

社員証・名刺・社内システムの更新

身上変更届の承認後、ITサポート（内線#3000）に社員証の再発行を依頼。名刺は総務部（内線#1500）へ。メールアドレス変更が必要な場合はIT部門へ別途依頼してください。

Moving Procedure

引越し・住所変更の手続き

転居後1週間以内に、通勤経路・住所変更の手続きが必要です。通勤手当が変更になる場合は翌月から反映されます。

1📝

Apply

HR Coreで「住所変更届」と「通勤経路変更申請」を提出

新住所・新通勤経路・最寄り駅・定期代を入力。経路は最短ルート（Google Mapsのスクリーンショットを添付）を申告してください。

❌ 住所変更の届出を怠ると、緊急連絡が届かない・住民税の特別徴収がずれる等のトラブルが起きます。転居後速やかに届出してください。
2🎫

Transport Card

Suica/PASMOの定期区間を変更・証明書を提出

定期区間変更後、定期券（または利用区間の証明）の写真をHR Coreに添付します。翌月給与から新通勤手当が適用されます（月末5日までの申請で翌月反映）。

Parental Leave

産休・育休申請

産前産後休業・育児休業の申請フローです。できるだけ早めの申請をお願いします。

📅育休申請：開始1ヶ月前まで 💰社会保険料免除あり 🎁出産祝い金（5万円）
1💬

Consult HR

妊娠が確認されたら、まず人事担当者に相談

産前何週から休みたいか・育休期間の希望を人事担当者と相談します。個別の事情（つわり・体調）に応じた対応もご相談ください。相談内容は上長には共有しません（本人許可がない限り）。

2📝

Submit Form

産前産後休業申請・育児休業申請をHR Coreで提出

母子健康手帳（写し）・出産予定日証明書を添付します。育休の開始・終了予定日を入力します（後から変更可能）。

📌 出産祝い金（5万円）はこのフォームから同時申請できます。
3🏥

Benefits Setup

社会保険料免除・育休給付金の手続き

人事担当者が代理で社会保険事務所に申請します（本人の作業不要）。育休給付金（雇用保険）の受取口座等は書類で確認します。

4🏠

Return Plan

復帰3ヶ月前にキャリア面談を実施

復帰希望日・勤務形態（時短・フルタイム・テレワーク等）を相談します。保育園・育児状況に応じて柔軟に対応します。

Benefits Application

福利厚生各種申請

テレワーク補助・自己啓発支援・慶弔金等の申請フロー一覧です。

テレワーク補助（年5万円）

購入から3ヶ月以内に申請。領収書添付必須。対象品目あり。

自己啓発支援（年20万円）

受講・受験前に事前申請が必要。事後申請は不可。

人間ドック補助（全額）

提携クリニックに予約 → 受診 → 領収書提出で全額補助。

慶弔金（結婚10万・出産5万）

事由発生から3ヶ月以内に申請。証明書類の提出が必要。

💡 すべての申請は HR Core の「福利厚生申請」メニューから行います。不明点は人事担当者（内線#2100）にご連絡ください。
