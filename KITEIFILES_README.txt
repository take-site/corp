規程PDFの英語ファイル名について
================================

現在、規程・制度ページ(regulations.html)のPDFリンクは
「kiteifiles」フォルダ内の日本語ファイル名を参照しています。

PDFを英語ファイル名で運用する場合：
1. ひとつ上のフォルダの kiteifiles 内のPDFを次のようにリネームしてください。

   就業規則.pdf → employment-rules.pdf
   給与規程.pdf → salary-regulations.pdf
   育児・介護休業規程.pdf → childcare-nursing-leave.pdf
   福利厚生規程.pdf → benefits-policy.pdf
   情報セキュリティ規程.pdf → information-security-policy.pdf

2. regulations.html 内のリンクを上記の英語ファイル名に書き換えてください。
   （../kiteifiles/就業規則.pdf → ../kiteifiles/employment-rules.pdf など）

※ kiteifiles はこの files フォルダのひとつ上にあります。
  例：サイト作成/kiteifiles/
