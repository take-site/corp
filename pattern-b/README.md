# パターンB — 絵文字なし版（シンプル表示）

このフォルダは、本サイトの**別パターン**です。  
ワンポイントで使っていた絵文字を、テキスト・記号（・— など）に置き換え、より落ち着いたトーンで統一しています。

## 使い方

- **トップページ**: `index.html` をブラウザで開く
- 採用サイト・社内ポータルとも、同じ階層のHTML同士でリンクしているため、このフォルダ内だけで完結して閲覧できます

## 元サイトとの違い

- 元サイト（ひとつ上の階層）: 絵文字をワンポイントで使用
- 本パターン（pattern-b）: 絵文字を廃止し、「・」「—」や「検索」「PDF」などのテキスト、またはステップ番号に統一

構成・レイアウト・文言は同じです。デザインのトーンだけを変えています。
