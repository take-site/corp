勤怠

8件
Q

有給休暇はいつから取得できますか？取得手続きを教えてください。

›

Answer

入社後6ヶ月が経過し、出勤率が80%以上の場合に10日間の有給が付与されます。その後、1年ごとに日数が増え、最大20日（勤続6.5年以上）まで付与されます。

申請は 勤怠管理システム（HR Core） から、希望日の3営業日前までに申請してください。承認後にメール通知が届きます。

⚠️ よくある失敗：前日・当日の申請は緊急の場合を除き認められません。計画的な事前申請をお願いします。

最終更新：2026年1月15日

Q

残業申請はいつまでに行えばよいですか？事後申請はできますか？

›

Answer

残業は原則として事前申請制です。当日17時までにHR Coreで申請し、上長の承認を得てから残業を開始してください。

緊急の場合のみ翌営業日中の事後申請が認められますが、月2回以上の事後申請は上長からの指導対象となります。

❌ やりがちなNG：申請なしで残業すると、残業代が不支給になる場合があります。必ず承認を得てから残業を始めてください。
残業申請の手順を見る →

最終更新：2025年11月1日

Q

フレックスタイム制での打刻ルールを教えてください。

›

Answer

始業・終業は HR Core でのICカード打刻または端末ログイン/ログアウトで自動記録されます。コアタイム（10:00〜15:00）中は必ず勤務してください。月の総労働時間が所定時間（原則160時間）を満たせばOKです。

⚠️ 打刻忘れが月3回以上続くと、勤怠評価に影響する場合があります。気づいた時点ですぐに上長に報告してください。

最終更新：2025年9月1日

Q

リモートワーク中の勤怠管理はどうすればいいですか？

›

リモートワーク日は事前に HR Core で申請が必要です（前週金曜17時まで）。当日のPCログイン/ログアウトが自動的に打刻として記録されます。外出が必要な場合は「外勤」として申請してください。

Q

特別休暇（慶弔休暇）の取得条件と日数を教えてください。

›

Answer

以下の特別休暇が取得できます（有給）：

結婚休暇：本人5日、子の結婚2日
忌引休暇：配偶者・子5日、父母3日、兄弟姉妹・祖父母1日
出産付添：2日
転勤：1日

取得には、戸籍謄本・案内状などの証明書類を申請から2週間以内に人事担当者へ提出してください。

⚠️ 証明書類の提出を忘れると、特別休暇が有給休暇に振り替わることがあります。

最終更新：2025年4月1日

Q

副業・兼業はできますか？届出は必要ですか？

›

Answer

2024年10月より副業・兼業が原則許可されています。ただし、以下の要件を満たし、事前に届出が必要です：

・本業に支障をきたさないこと
・競合他社・取引先での就業でないこと
・週20時間以下であること

HR Coreの「副業・兼業申請」フォームから申請し、承認を得てから開始してください。

⚠️ 届出なしで副業を行った場合、懲戒対象となることがあります。

最終更新：2024年10月1日

Q

退職する場合、何ヶ月前に申し出が必要ですか？

›

Answer

就業規則上の退職申出期間は原則2ヶ月前です。引き継ぎ期間を考慮し、早めにご相談ください。申出方法は所属長への口頭報告 → 退職申出書の提出（HR Core）の順で手続きをお願いします。

最終更新：2025年4月1日

Q

リモートワーク環境整備補助金の申請方法を教えてください。

›

Answer

在宅勤務環境整備のため、年間上限5万円の補助が受けられます。対象品目：デスク・椅子・モニター・キーボード・WEBカメラ・ルーター等。

手順：①購入 → ②領収書保管 → ③HR Coreで「テレワーク補助申請」 → ④領収書スキャンを添付 → ⑤翌月給与に加算

⚠️ 購入から3ヶ月以内に申請しないと受付できません。領収書は必ず保管してください。
❌ 対象外：スマートフォン、家電（冷蔵庫等）、個人用ゲームデバイス
申請フロー詳細を見る →

最終更新：2025年10月1日

Q

育児休業（育休）はいつまでに申請が必要ですか？

›

Answer

育児休業の申請は、開始予定日の1ヶ月前まで（出産予定日が分かり次第できるだけ早めに）行ってください。

産前は産後休業（産前6週・産後8週）の申請も別途必要です。HR Coreで手続きを行い、母子手帳の写し等の証明書類を提出してください。

育休期間中は社会保険料が免除され、育児休業給付金（雇用保険）が受給できます。

産休・育休の手続きフローを見る →

最終更新：2026年2月1日

Q

自己啓発支援（資格取得補助）の対象と申請方法を教えてください。

›

Answer

年間上限20万円で、業務に関連する資格・研修・書籍の費用を補助します。資格試験費用は合格時に全額支給（不合格時は50%）。

申請タイミング：受講前に「自己啓発申請」 → 上長承認 → 受講 → 領収書提出 → 翌月加算

⚠️ 事前申請なしで受講・購入した場合は補助の対象外となります。必ず受講前に申請してください。

最終更新：2025年4月1日

給与・賞与

5件
Q

給与明細はどこで確認できますか？

›

Answer

給与明細は HR Core「給与・勤怠」タブ から確認できます。毎月25日に当月分を公開します（25日が休日の場合は前営業日）。紙の明細は発行しておりません。PDFでダウンロードが可能です。

⚠️ 内容に誤りがある場合は当月中に人事担当者へご連絡ください。翌月以降の修正は対応が困難になります。

最終更新：2025年4月1日

Q

賞与の支給時期と計算方法を教えてください。

›

Answer

賞与は年2回、7月（夏季）と12月（冬季）に支給されます。

計算方法：月額基本給 × 係数（0〜6.0）。係数は会社業績（50%）と個人評価（50%）の組み合わせで決まります。評価期間は上期1〜6月・下期7〜12月です。

最終更新：2025年4月1日
